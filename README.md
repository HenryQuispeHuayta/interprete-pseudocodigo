# **Interprete-Pseudocodigo**

## **Pre-requisitos**

Se requiere Python, que se puede descargar desde [aquí](https://www.python.org/downloads/).

## **Ejecución**

La primera forma de ejecutar el programa es a través de la shell.

```
python shell.py 
```

También se puede ejecutar un archivo específico con el siguiente comando:

```
python shell.py ruta-archivo/ejemplo.inf
```

Estos archivos deben tener la extensión `.inf`.
